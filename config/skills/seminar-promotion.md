# スキル: Claude Codeセミナー集客投稿

## 目的
Claude Code無料セミナーへの集客を目的としたX投稿の生成ルール。

## セミナー情報(固定)

- **タイトル**: 「話題のClaudeって何がすごいの？」
- **URL**: https://claude-code-seminar-lp.vercel.app/intro/
- **主催**: 株式会社ノヴァリス
- **講師**: 田中俊輔(AI事業責任者/営業経験からAI独学)
- **価格**: 無料
- **特典**: 録画アーカイブ配信、講師による個別相談
- **対象**:
  - AIが「自分には関係ない」と思っている人
  - ChatGPT経験者で更に学びたい人
  - PC利用者全般(専門知識不要)

## 主要な訴求内容
- PCの自動操作
- スマホからのリモート制御
- 複数ファイルの同時生成
- メール/カレンダー/Web検索の自動化
- Webサイト開発
- 定時実行タスク

## 投稿パターン(必ずA型 = 事件/スクープ型)

### 構造
1. **冒頭フック**: `【朗報】` `【速報】` `【衝撃】` のいずれかで開始
2. **具体ディテール**: 箇条書き3-4項目で驚きを盛る
3. **ChatGPTとの差別化**: 既存ユーザーを動かす一言
4. **CTA**: ▼ + 「無料セミナーで実演(録画あり)」+ URL

### 字数目安
- 本文: 150-220字
- URL: 23字(t.co短縮後)
- 合計: 280字以内厳守

## 実証済み3フック(使い回し可)

### フックA: PC全自動の衝撃 (推奨)
- **ターゲット心理**: 「AI興味ない」勢の好奇心を刺激
- **差別化**: 「答えるAI」vs「代わりに作業するAI」
- **CSV参考元**: masahirochaen (6236 likes)

### フックB: Anthropic急成長
- **ターゲット心理**: ChatGPT派に「乗り遅れ」危機感
- **差別化**: 企業導入事例(メルカリ/楽天)で権威付け
- **CSV参考元**: masahirochaen企業戦略系

### フックC: スマホ1行で全自動
- **ターゲット心理**: 「楽したい」「ラクラク」訴求
- **差別化**: 専門知識ゼロでOKの明言
- **CSV参考元**: 0x__tom (スマホ連携系)

## 採用語彙(lift_words)

高反応が出やすいキーワード:
- 【朗報】【速報】【衝撃】【悲報】
- 「自分には関係ないと思ってた」
- 「ChatGPTとの違い」
- 「代わりに作業するAI」
- 「メルカリも導入」
- 「録画あり」「無料」「専門知識ゼロ」

## 避ける語彙(bad_patterns)

- ハッシュタグ (#AI, #プログラミング 等) — Views半減
- 数字で始める (「1. はじめに」等) — ヒット率5%まで落ちる
- 「すごい」「やばい」のみの中身なし感想
- 「ぜひ」「ご興味のある方は」系の弱いCTA

## 禁止事項

- 同じフックを3投稿連続で使わない(ローテーション)
- URL を文頭/文中に置かない(必ず末尾)
- セミナー情報の改変禁止(URL/講師名/価格など)
